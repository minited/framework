Overlays component demo

General-purpose drawer/offcanvas, action menus, and popover cards — all using the Popover API.

Drawer / offcanvas

  • Uses the Popover API — add popover="manual" to the element.
  • Click the backdrop or the close button to dismiss.
  • Add .drawer--end for a right/inline-end panel.
  • Override width: style="--drawer-panel-width: 480px".

Action menu

CSS-only version uses :focus-within. Click trigger to open; click away to close. For the Popover API version, add popover to the list and wire up a button.
