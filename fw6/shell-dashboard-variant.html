Dashboard shell

Combines a sticky sidebar, a sticky topbar, and a scrollable main area using CSS Grid. The sidebar spans the full viewport height; the topbar sits above main only.

How to use
  • Three children: .dashboard-shell__sidebar, .dashboard-shell__topbar, .dashboard-shell__main.
  • Override sidebar width: style="--sidebar-width: 20rem" on .dashboard-shell.
  • On mobile (< 1024px) the sidebar hides; the topbar spans full width.
  • The topbar uses backdrop-filter: blur for a frosted glass effect over scrolling content.
