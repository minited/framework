Standard content

A contained section with text that follows the container width.

Use this when you want ordinary page copy, explanatory text, or supporting content that aligns with the rest of the page layout.

This section is deliberately not wrapped in .prose. It shows the default content pattern where text fills the container width. This is often the best choice for marketing pages, services pages, and many general website sections.

If you want a narrower reading measure for longer text, use the prose pattern in the next section instead.

Prose

A narrower reading width for article-style content.

The prose component is useful when the page includes longer reading text and you want a more comfortable line length. It keeps paragraphs from stretching too wide on large screens.

Use this pattern for blog posts, explanations, long descriptions, case studies, or any page where readability matters more than filling the full container width.

Students should choose between standard contained content and prose based on purpose, not just habit.

FAQ

Optional accordion example.

Do I need every component on every page?
No. The whole point of the modular system is to include only what each page actually uses.
Should I always use prose for text?
No. Use prose for longer reading text. Use normal contained content when you want the copy to align with the rest of the page width.
