Dialog

Dialog / modal component demo

Uses the Popover API — no JavaScript required. Open with popovertarget, close with popovertargetaction="hide". Clicking the backdrop or pressing Escape also closes.

Sizes

Usage notes

  • Add popover to any element to make it a modal. Wire a trigger with popovertarget="id".
  • Close buttons use popovertarget="id" popovertargetaction="hide" — no JavaScript.
  • Clicking the backdrop (outside the modal) or pressing Escape dismisses automatically.
  • Size variants: .modal--sm (360px), default (540px), .modal--lg (780px), .modal--full.
  • The Popover API does not trap focus. Add a JS focus-trap utility if needed for complex forms.

Small modal

This is a small modal — --modal-width: 360px. Good for short confirmations or alerts.

Default modal

Default width is 540px. The body scrolls independently when content overflows.

The header and footer stay fixed while the body scrolls.

Large modal

Large modal — --modal-width: 780px. Use for complex content, data tables, or media previews.

Override width directly: style="--modal-width: 1000px" on the element.
