Shell

Sidebar + Main layout

The .shell primitive creates a sticky sidebar beside a scrollable main area. On mobile (< 768px) it stacks vertically.

How to use
  • Wrap .shell__sidebar and .shell__main in .shell.
  • Override sidebar width: style="--sidebar-width: 20rem" on .shell.
  • The sidebar is sticky and scrolls independently from main.
  • See shell-dashboard-variant.html for the full dashboard shell with a topbar.
  • See shell-master-detail-variant.html and shell-filter-results-variant.html for other layout patterns.
