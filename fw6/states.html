Quick reference

Class Effect Use case
.is-active Accent colour Hover-like emphasis without interaction
.is-selected Tinted background + accent text + bold Chosen item in a list or grid
.is-current Accent colour + bold + no pointer events Current page in navigation
.is-disabled 45% opacity + no pointer events Unavailable action or item
.is-readonly Default cursor + no pointer events Visible but non-interactive
.is-loading 65% opacity + no pointer events Waiting for an async action; pair with a spinner
.is-error Error border + error text colour Failed state
.is-success Success border + success text colour Completed state
.is-warning Warning border + warning text colour Needs attention
.is-info Info border + info text colour Informational context
