Starter section

A standard content section that fills the container.

Use this pattern when you want ordinary section text and supporting copy to follow the main page width rather than a narrower reading measure.

This is a good default for service pages, landing pages, and general website sections. The container keeps the layout tidy, while the text is allowed to use the available width inside that container.

Delete this copy and replace it with your own content. If you want narrower long-form reading text instead, add the prose component and wrap your text in a .prose container.
